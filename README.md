**EventHive** is a modern, responsive web application designed to streamline the event management lifecycle. It allows users to discover events and register instantly, while providing organizers with a powerful dashboard to track analytics and manage attendee check-ins.

## ğŸ§ª How the Mock Backend Works

This project uses **MSW (Mock Service Worker)** to intercept network requests generated by the application.
- **No Backend Required**: All "database" operations happen in the browser's memory.
- **Realistic Latency**: An artificial delay (600ms) is added to all requests to simulate real-world network conditions.
- **Note**: Since data is stored in memory, refreshing the page will reset the application state to the default seed data.
